# AI-Assisted Adaptive Speech Therapy Platform

A comprehensive full-stack platform designed to bridge the gap between in-clinic sessions and at-home practice. Uses state-of-the-art AI (Whisper-small for ASR and Wav2Vec2 for Emotion Recognition) to analyze patient speech and adapt weekly therapy plans dynamically.

## Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS v4, React Router DOM, Chart.js, Clerk Auth, Shadcn UI
- **Backend**: FastAPI (Python), SQLAlchemy ORM (asyncpg), PostgreSQL, Clerk Backend API
- **Machine Learning**: HuggingFace Transformers (Whisper-small, Wav2Vec2), PyTorch, Librosa

## Getting Started

### Prerequisites
- Node.js (v18+)
- Python (v3.10+)
- PostgreSQL installed and running locally
- Clerk Account (for Auth keys)

### 1. Database Setup
Create a PostgreSQL database named `speech_therapy`:
```bash
createdb speech_therapy
```

### 2. Backend Setup (FastAPI)
Navigate to the server directory:
```bash
cd server
python -m venv venv
# Windows: venv\\Scripts\\activate | Mac/Linux: source venv/bin/activate
pip install -r requirements.txt
```
Copy `.env.example` to `.env` and fill in your database credentials and Clerk keys.

Initialize the database schema:
```bash
python seed.py
```
Start the backend server:
```bash
uvicorn main:app --reload
```
The API runs at `http://localhost:8000`.

### 3. Frontend Setup (React/Vite)
Navigate to the client directory:
```bash
cd client
npm install
```
Copy `.env.example` to `.env.local` and fill in your Clerk Publishable Key.

Start the frontend server:
```bash
npm run dev
```
The App is accessible at `http://localhost:5173`.

## Architecture Details
- **Therapist Dashboard**: Accessible to users with `role: therapist`. Contains tools for patient management, monitoring insights generated by the AI, and adjusting therapy assignments.
- **Patient App**: A mobile-first interface guiding patients through gamified daily tasks (Word Repetition, Sentence Reading). Records audio and submits it to the FastAPI backend for scoring.
